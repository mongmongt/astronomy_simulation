도플러 효과 & 수식적 배경

1. 도플러 효과 (Doppler Effect)
파원과 관측자가 서로 다가가거나 멀어질 때, 관측자가 느끼는 파동의 진동수(또는 파장)가 변화하는 현상입니다. 우주론에서는 지구로 다가오는 천체는 파장이 압축되는 청색편이(Blueshift)가, 지구로부터 멀어지는 천체는 파장이 늘어나는 적색편이(Redshift)가 관찰됩니다.

2. 수학적 원리 (시선속도의 중요성)
도플러 편이를 유발하는 핵심 속도는 은하의 전체 속도($v$)가 아니라, 지구와 은하를 잇는 직선상의 방향 성분인 시선속도(Radial Velocity, $v_r$)입니다. $$\vec{v} = (v_x, v_y, v_z)$$ $$v_r = \vec{v} \cdot \hat{r}$$ 여기서 $\hat{r}$은 지구에서 은하를 바라보는 단위 방향 벡터입니다.

3. 파장 변위 공식
빛의 속력 $c \approx 300,000\text{ km/s}$에 비해 은하의 속도 $v_r$이 충분히 작을 때 비상대론적 도플러 공식은 다음과 같습니다. $$z = \frac{\Delta\lambda}{\lambda_0} = \frac{\lambda - \lambda_0}{\lambda_0} = \frac{v_r}{c}$$ $$\lambda = \lambda_0 \left(1 + \frac{v_r}{c}\right)$$

  • $v_r > 0$ (지구로부터 멀어지는 중) $\rightarrow \lambda > \lambda_0$ : 적색편이
  • $v_r < 0$ (지구로 가까워지는 중) $\rightarrow \lambda < \lambda_0$ : 청색편이

4. 3차원 방향 각도의 수학적 연동 제약
3차원 유클리드 좌표계에서 임의의 방향 벡터가 $x, y, z$ 축의 양의 방향과 이루는 각도를 방향각 $\alpha, \beta, \gamma$라 할 때, 기하학적으로 항상 다음 관계식을 엄격히 만족해야만 합니다. $$\cos^2\alpha + \cos^2\beta + \cos^2\gamma = 1$$ 본 시뮬레이션에서는 사용자가 임의의 한 슬라이더를 변경하면 이 법칙이 항상 지켜지도록 다른 두 변수가 부드럽고 비례적으로 자동 동기화 계산됩니다.
